# Tech Economist — techeconomist.in
## Setup
Save your tool HTML files into `tools/`:
1. Markov Chain → `tools/markov.html`
2. RCA Analyser → `tools/rca.html`
3. LDV Regression → `tools/regression.html`
Citation Manager is already built (`tools/citation.html`).
## Deploy
`git init && git add . && git commit -m "init" && git push`
Settings → Pages → main branch → Custom domain: techeconomist.in
